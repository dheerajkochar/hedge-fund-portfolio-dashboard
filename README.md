
# Hedge Fund-Style Portfolio Dashboard

This project is a SQL-first stock portfolio analytics system with:

- FIFO Realized P&L
- Unrealized P&L
- Equity Curve
- Drawdown
- Daily P&L
- Trade Ledger
- Stock Performance Charts

## Tech Stack
- PostgreSQL
- Python
- Streamlit
- Pandas
- Plotly

## Features
- SQL-based FIFO logic
- Backfilled price ingestion
- Simulated trade engine
- Real-time dashboard
